# civic-tech-experimental-field-find-babycare-backend
本案為「112年數位發展部公民科技試驗場域示範案」由「一起加油陣線」團隊開發的「公托申請數位流程優化」透過與臺中市政府社會局的合作，提供既有公托申請流程解方，開發一申請公托的整合平台，提供申請、查詢、審核及管理等功能。本案為示範案例尚未正式上線提供服務。
